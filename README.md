## Task Title
Phishing Email Detection & Awareness System

## Objective
To analyze phishing emails and create a professional awareness report to help users identify and avoid phishing attacks.

## Tools Used
- Phishing email samples
- Google Email Header Analyzer / Manual header analysis
- MXToolbox
- Browser inspection tools (hover over links)
- Canva / Google Docs / Microsoft Word

## Analysis Approach
1. Collected phishing email samples (offline / simulated)
2. Analyzed email content for phishing indicators:
   - Suspicious sender
   - Urgency or fear tactics
   - Malicious links or attachments
3. Performed email header inspection (manual)
4. Inspected links and domains for legitimacy
5. Classified email risk levels (Phishing / Suspicious / Safe)
6. Created awareness and prevention guidelines
7. Documented all findings in a professional PDF report
8. Included screenshots of emails, headers, and link analysis in `/evidence` folder
